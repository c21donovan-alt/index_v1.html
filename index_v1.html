  4. Enable GitHub Pages

    • Go to the repo’s Settings → Pages (on the sidebar).

🌿 Option 2: Host for Free on Netlify

⚙️ Step-by-step Guide

  1. Go to https://www.netlify.com/ and create a free account.

  2. Once logged in, choose “Add new site → Deploy manually.”

  3. Drag and drop your index.html file (and any additional files) into the upload area.
